# Regla de Idioma: Español

Este proyecto requiere que toda la comunicación con el usuario y la documentación generada (a menos que se especifique lo contrario para código técnico) se realice en **Español**.

## Instrucciones para el Asistente:
1. **Siempre responder en español**.
2. Evitar cambiar al inglés de forma automática.
3. Si se olvida esta regla, el usuario recordará "en español siempre".
4. Toda la documentación nueva en la carpeta `docs/` debe ser en español.
